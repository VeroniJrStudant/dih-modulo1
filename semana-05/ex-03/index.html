[M1S05] - Ex 3 - Crie uma classe Endereco

Crie uma nova classe para representar as informações de um endereço. A classe Endereco possui os seguintes atributos: logradouro, numero, cidade, estado, pais, cep. No constructor() desta classe, verifique se todos os valores recebidos por parâmetro são do tipo 'string' (utilize typeof). Caso algum não seja do tipo 'string', lance (throw) uma exceção com a seguinte mensagem: "Endereço inválido!". Ex.: throw 'Endereço inválido!' Dica: https://www.w3schools.com/js/js_errors.asp
