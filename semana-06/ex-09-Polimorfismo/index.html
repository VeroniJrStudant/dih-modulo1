Ex 09 - Poliformismo

No arquivo ‘Funcionario.js’, semelhante à classe Pessoa, implemente na classe Funcionario um método ‘imprime’, mas desta vez, faça com que ele mostre no console o nome e o cargo da instância de Funcionario (ex.: “Livia - gerente”). Aproveite o mesmo array com as três instâncias de Funcionario criadas no exercício anterior e implemente um loop forEach que, para cada item do array, execute o método ‘imprime’, exibindo os respectivos nomes e cargos. Dica: item => item.imprime()

Para visualizar o exercício pressione "F12"
